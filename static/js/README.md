## 项目结构

- **apps** app组件
- **bases** 基础服务
	- **components** 全局组件
	- **directives** 全局指令
	- **filters** 全局过滤器
	- **mixins** 混入文件
	- **partials** html片段
	- **transitions** 全局动画钩子函数
- **config** 全局配置信息
- **core** 核心代码
	- **error** 错误页面处理
	- **home** 首页
	- **router** 路由控制
	- **service** 基础服务
- **module** 组件池
